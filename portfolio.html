EVORP

EVORP is a roleplaying gamemode for the game "garry's mod"

The server attracted almost 500 players a day and the community reached 25000+ active members.



AsgardDEV

AsgardDEV was designed from scratch to emulate a MapleStory server (v110.2) in C#.

It involved packing sniffing, analyzing & decoding, understanding the structure and programming the server response based on the data.

It used the following library: MapleLib
GitHub: AsgardDEV

Chat Server & Client

This is a instant messaging application made entirely in Java. It uses the POJ Sockets and client server communication takes place in plain unencrypted strings.

Each string starts with a command such as #pm, #whisper, #disconnect, #alert, #connected, etc that controls client & server behaviour.

GitHub: JAVAMessaging


SharpStory

SharpStory was a v83 MapleStory server emulator that was programmed in Java, used MySQL as a database and PHP as it's online management system.

GitHub: SharpStory
